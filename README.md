# 📊 E-commerce Sales Analysis Dashboard

## 🚀 Project Overview
This project focuses on analyzing an E-commerce dataset (1200+ records) to extract meaningful insights and build an interactive dashboard using Excel.

The goal of this project is to simulate real-world business scenarios and help stakeholders make data-driven decisions.

## ⚙️ Feature Engineering
Additional columns were created to enhance analysis:

- **Profit Category** → Classified as High Profit / Low Profit  
- **Month-wise Analysis** → Extracted month from order date  
- **Discount Impact** → Categorized as High Discount / Low Discount  
- **Profit Margin (%)** → Calculated profitability ratio

## 💡 Key Insights
- Electronics category contributes the highest share of revenue  
- Certain cities (like Hyderabad & Delhi) drive higher sales  
- High discounts often reduce overall profit margins  
- Repeat customers contribute significantly to total revenue
